# Tripling Beta

We are developing a project-level method to estimate a plausible range of annual capacity additions from 2026 to 2030 based on GEM's capacity in development data. The starting point is the GIPT dataset of individual power projects, some with known intended commissioning/start years and some without. The aim is to move from a simple "known pipeline by stated year" view toward an illustrative, risk-adjusted outlook that accounts for uncertainty around project timing, delays, cancellations, and undated projects.

This is not intended to be a precise forecast of individual project delivery. It is an assumption-based simulation designed to show the range of additions that could plausibly materialise from the current project pipeline under transparent assumptions.

Read more on current set-up in the [methodology document](methodology.docx).
